when linking images to a webpage you can either link to an image on a website using the image address or you can link from an image stored locally. Linking from another website is generally seen as bad practice as the image is not under your controll e.g. the other website might remove or change the image and that will effect your page

the tag to show this image locally is called <img src="localimageaddress.jpg" />

the address you use is relative it will be relative to the html file
